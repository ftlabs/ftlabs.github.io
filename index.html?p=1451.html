Android application developer

Join us as an Android application developer to help build great applications and websites for a wide variety of platforms and audiences. You’ll be working with skilled developers, UX and test engineers, and end-users to deliver a great experience to those who choose to interact with our apps on the Android platform.

This page describes the Android application developer role specifically. For general recruitment information and a guide to what working at FT is like, or for details of our other jobs, please see our main jobs page.

Responsibilities may include:

  • Developing applications using the Android SDK, for large and small screen devices running Jelly Bean, ICS and Gingerbread
  • Integrating native Android code with web views running HTML5 web applications
  • Participation in the Android community
  • Working with the Android team at Google and hardware manufacturers
  • Liaising with end users to gather and aggregate bug reports
  • Monitoring Android market feedback and reacting swiftly to user demand
  • Writing and maintaining unit tests

Required skills

  • A good working knowledge of Java
  • Experience with Android development
  • Entry level knowledge of PHP, JavaScript, HTML and CSS, including object oriented concepts
  • Working knowledge of Linux command line tools and version control concepts
  • Excellent quality of written and spoken English

Desirable skills

  • More advanced JavaScript or PHP knowledge
  • Credited sole author of applications in the Android Market (please send us links!)

Salary

We offer salaries of £25,000 – £45,000 for Android application developer roles
